# Quantum Versions of Classical Algorithms

This repository explores the intersection of quantum computing and machine learning by implementing quantum counterparts to standard classical algorithms. The models are built utilizing **PennyLane**, a powerful library for differentiable quantum programming.

## 🚀 Implemented Algorithms

The repository is organized into folders, each containing Jupyter Notebooks for the following quantum algorithms:

* **K-Nearest Neighbors (KNN):** Quantum distance estimation techniques to calculate the proximity between data points.
* **Clustering:** Quantum-adapted clustering approaches (such as Quantum K-Means) utilizing quantum circuits for centroid calculation and assignment.
* **Support Vector Machine (SVM):** Implementations of Quantum Kernel Methods to map classical data into high-dimensional quantum Hilbert spaces for classification.
* **Quantum Markov Models:** Exploration of Markov processes through the lens of quantum state transitions.

## 🛠️ Technologies & Libraries

The project relies on the following Python ecosystem:

* **PennyLane:** Core framework for building, training, and simulating quantum circuits.
* **NumPy & SciPy:** For efficient linear algebra, matrix operations, and mathematical computations.
* **Pandas:** For structural data manipulation and preprocessing.
* **Scikit-Learn:** For dataset generation, classical baseline comparisons, and standard ML utility functions.
* **Matplotlib:** For visualizing data distributions, decision boundaries, and model performance.
