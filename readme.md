# Quantum machine learning

In this repository I have made use of pennylane library which is used for quantum computing. The classical machine learning algorithms for which I have written quantum versions are:

1. KNN (K nearest neighbour)
2. Clustering
3. SVM

The packages of python which are used by me in this project are:

```text
1. Numpy
2. Pandas
3. Matplotlib
4. Scipy
5. Pennylane
6. Scikit learn
```
